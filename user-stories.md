# User Stories

## 👨‍💼 Admin User Stories
- As an **admin**, I want to add participant details so that I can manage support services efficiently.
- As an **admin**, I want to add support worker profiles so that I can assign suitable workers.
- As an **admin**, I want to create shifts so that participant services are scheduled correctly.
- As an **admin**, I want to assign workers to shifts so that support is delivered on time.
- As an **admin**, I want to view reports so that I can track shift completion.

---

## 👷 Support Worker User Stories
- As a **support worker**, I want to view my assigned shifts so that I know my work schedule.
- As a **support worker**, I want to update shift status so that admin can track progress.
- As a **support worker**, I want to add case notes so that participant support details are recorded.

---

## 👤 Participant User Stories
- As a **participant**, I want to view my support schedule so that I know upcoming services.
