# CareRoster – NDIS Roster Management System

## 📌 Overview
CareRoster is a web-based NDIS Roster Management System designed to help service providers manage participants, support workers, shift scheduling, and case notes in one platform.

This project is being developed as a student portfolio project to demonstrate both **software development** and **business analysis** skills.

---

## 🎯 Project Objectives
- Manage participant records
- Manage support worker profiles
- Create and assign shifts
- Prevent double booking
- Record case notes
- Generate shift and support reports

---

## 👥 User Roles
### Admin
- Add participants
- Add support workers
- Create shifts
- Assign workers
- View reports

### Support Worker
- View assigned shifts
- Update shift status
- Add case notes

### Participant
- View support schedule
